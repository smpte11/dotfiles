# Pure flavor for yazi

A minimal flavor inspired by the [Pure](https://github.com/sindresorhus/pure)
zsh prompt: near-monochrome, muted-grey chrome, a single magenta accent, and a
blue path.

## How it works

`flavor.toml` uses **only ANSI color names** (`magenta`, `blue`, `darkgray`, …)
and `reset`. It defines no hex, so the entire UI inherits whatever palette your
terminal provides — exactly how Pure itself stays palette-agnostic. In this
dotfiles setup the palette comes from ghostty, driven by `active_theme` in
`.chezmoidata.toml`.

The one exception is `tmtheme.xml` (preview-pane syntax highlighting). tmTheme
files require hardcoded hex, so it is generated by chezmoi from `active_theme`
and stays in sync when you switch themes.

## Install

Managed by chezmoi. `chezmoi apply` writes it to
`~/.config/yazi/flavors/pure.yazi/` and selects it via `~/.config/yazi/theme.toml`.
